# Legacy

This directory preserves non-primary code paths and experiments that should not be deleted during the migration.

## Current Contents

- `static-web-prototype`: the earlier static HTML/CSS/JS frontend experiment
- `plugin-experiment`: the early QML plugin prototype
- `pyqt-overlay`: a preserved copy of the earlier root-level PyQt overlay file

## Usage Rule

Treat these files as reference material, not as the primary place for new feature work.
