Maestro v0.1.0

AI-assisted composition for MuseScore.

Prompt a musical change, optionally hum an idea, and let Maestro turn that request into score edits for a live MuseScore session.

Download for macOS ZIP fallback

The macOS beta is unsigned and not notarized. On first launch, right-click Maestro.app and choose Open.

Manual setup for Windows, Linux, or source-based macOS

  1. Clone the repo and create a virtual environment.
  2. Run python -m pip install -r requirements-desktop.txt.
  3. Install the MuseScore plugin with maestro-install-plugin install.
  4. Enable Maestro Plugin in MuseScore and run it from Plugins > Maestro.
  5. Launch the app with python maestro_gui.py.

Full instructions: README.md

How AI-assisted composition works

Maestro takes your prompt or hummed melody, asks a language model for score-edit code, converts that result into score operations, and applies those operations through a live MuseScore plugin bridge. It is a beta co-writing tool, not an autonomous or reliability-guaranteed composer.
